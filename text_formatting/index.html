Main Formatting

Bold

To write something in bold you have 2 ways.

You can write using B (not semantic)

Or you can write using the strong tag (semantic)

Italic

To write something in italic we have two ways too

You can wite using the I tag (not semantic)

Or you can use the tag EM (semantic)

Mark

We can create a marked text using the tag Matk, for example: Here we have a marked text

Deleted Text

we can write a deleted text that can be read but not considered using the tag DEL

Inserted Text

We can emphasize a text using the that INS (semantic)

We can also underline some text using the tag U (not semantic)

Other Formating Types

Source Code / Preformatted

When we write a source code with no formatting, it is visually hard to understand, for example: document.getElementById('Teste')

to understand better, we can use the tag code: document.getElementById('Teste')

When we want to write a complete code, we can use the tag PRE with Code, for exmaple:

x = 10

if x > 5:
    print("x é maior que 5")
else:
    print("x é menor ou igual a 5")

Simple Quotes

To say something that another peron said you can use the tag Q (semantic), for example: As Beyoncé once said , I'm one of one, I'm number one, I'm the only one

Abbreviation

We can explain the meaning of a abbreviation with the tag ABBR, for example: HTML and CSS are part of an website
